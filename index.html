Build

  1. Get Cloud Commander and node-webkit.
  2. Unpack them to root app dir.
  3. Zip to cloudcmd-app.zip files: package.json, favicon.png
  4. Rename cloudcmd-app.zip to cloudcmd-app.nw
  5. Start with cloudcmd-app.bat
